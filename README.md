# Credit Scoring Model using Machine Learning

## Project Overview

This project predicts whether a customer is likely to become a credit risk using historical financial records.

The project is developed as part of the **CodeAlpha Machine Learning Internship**.

---

## Objectives

* Predict credit risk
* Compare multiple machine learning algorithms
* Perform feature engineering
* Build an end-to-end ML pipeline
* Deploy using Streamlit

---

## Dataset

Give Me Some Credit Dataset

https://www.kaggle.com/competitions/GiveMeSomeCredit

---

## Algorithms

* Logistic Regression
* Decision Tree
* Random Forest

---

## Evaluation Metrics

* Accuracy
